# AI Search Algorithms Collection

Welcome to the companion repository for exploring foundational search strategies in Artificial Intelligence. This repository serves as an educational resource for students and researchers to study tree and graph traversal methods.

## 📂 Contained Algorithms

| Category | Algorithm File | Description |
| :--- | :--- | :--- |
| **Uninformed Search** | `Breadth-First Search.py` | Explores shallowest nodes first using a FIFO queue. |
| | `Depth-First Search.py` | Explores deepest nodes first using a LIFO stack. |
| | `Depth-First Search Recursive.py` | Recursive implementation of standard DFS. |
| | `Depth Limited Search.py` | DFS with a predetermined depth limit to avoid infinite paths. |
| | `IDDFS.py` | Iterative Deepening DFS; combines DFS space efficiency with BFS completeness. |
| | `UCS.py` | Uniform Cost Search; expands nodes based on optimal path cost. |
| **Adversarial Search** | `MiniMax.py` | Decision-making algorithm used in two-player turn-based games. |
